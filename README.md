# E-Commerce Project (POO 2025-2026)

## Description
This is an N-Tier architecture console application for managing users, products, and orders.

## Architecture
- **BO:** Business Objects (Entities)
- **Data:** Repositories and File Storage (Binary)
- **Rules:** Business Logic (Services)
- **Exception:** Custom Error Handling
- **Tests:** Unit Testing Project (MSTest)
- **Program (ComercioEletronico):** Console UI

## Features
- User Authentication (Client)
- Product Management
- Order Processing
- Automated Documentation (Doxygen)
